How XP Works

XP (experience points) is earned through sessions, activities, quests, and daily streaks. XP accumulates to level you up, unlock new features, and climb the leaderboard.

Almost all XP passes through the global XP multiplier from the Skill Tree, so unlocking multiplier skills accelerates everything.

Key Concepts

Base XP — the raw amount before multipliers.
Global multiplier — applied to all XP sources via the Skill Tree.
Session multiplier — an additional multiplier that only applies to session completion XP, based on your level.

XP Sources

Sessions

Sessions are the biggest XP source. Each session type awards a base XP (via BonusXP), which is then scaled by the session level multiplier and a duration bonus.

Session Base XP

The raw base is min(2500, BonusXP) - (pauseCount × 100), floored at 0. Each pause costs 100 XP. Typical BonusXP values by difficulty:

Difficulty Typical Base XP Examples
Easy ~400 XP Morning Drift
Medium ~800 XP Random Drop, Gamer Girl
Hard ~1,200 XP Good Girls Don't Cum, Deep Dive
Extreme ~2,000 XP Bambi Time
Remote Control 200 XP Controlled by another user
Custom (Timeline) Varies Based on duration + enabled features

Duration Bonus

Sessions longer than 2 minutes award extra XP based on duration and level:

Duration Bonus = (minutes - 2) × (8 + level × 0.15)

Example: Level 50, 30-minute session
  = (30 - 2) × (8 + 50 × 0.15)
  = 28 × 15.5
  = 434 bonus XP

Final Session XP

Final = round(base × sessionLevelMultiplier) + durationBonus

The session level multiplier only applies to the base, not the duration bonus. The global Skill Tree multiplier is then applied on top of the final value.

Quests

Quests are a major XP source that scales with your level. See the Quests guide for the full list of available quests.

Quest XP = baseXP × (1 + level × 0.04) × betterQuestsMult × (1 + questStreak × 0.03)
Factor Effect
Level scaling +4% per level (multiplicative)
Better Quests skill 1.25x quest XP
Quest streak +3% per consecutive day of daily quest completion
Quest Type Base XP Range Scaling
Daily Quests 100 - 250 XP +4% per level
Weekly Quests 500 - 1,000 XP +4% per level

Pink Rush

Pink Rush

3x multiplier on everything for 60 seconds. The entire additive sum above is tripled.

Trigger: 50% chance every 10 minutes (checked by timer). Only while the Pink Rush skill is unlocked.
Duration: 60 seconds, then ends automatically.

Session Level Multiplier

This multiplier only applies to the session base XP (not the duration bonus, not activities). It kicks in at level 30 and scales up:

Maximum Multiplier

Maximum Multiplier

Without Pink Rush: up to ~2.1x (all Sparkle Boosts + max Streak Power + Night Shift or Early Bird)
With Pink Rush: up to ~6.3x (entire sum above, tripled)

These global multipliers stack on top of the session level multiplier. A Level 150+ player during Pink Rush + Night Shift could theoretically earn 5.0x × 6.3x = 31.5x the base session XP.

Leveling

XP Per Level

Each level requires an increasing amount of XP. The formula changes at each tier:

Level Range XP Per Level Formula
1 - 80 800 → 2,500 800 + (level - 1) × (1700 / 79)
81 - 100 2,500 → 4,000 2500 + (level - 80) × (1500 / 20)
101 - 125 4,000 → 6,000 4000 + (level - 100) × (2000 / 25)
126 - 150 6,000 → 10,000 6000 + (level - 125) × (4000 / 25)
151+ 10,000+ (exponential) 10000 × 1.03^(level - 150)

Cumulative XP

Reaching Level 50 requires roughly 62,000 total XP. Level 100 requires roughly 190,000 total XP. Level 150 requires roughly 490,000 total XP. Beyond 150, exponential growth means each level takes 3% more XP than the last.

Fair Play

Idle Suppression

To prevent AFK farming, passive XP sources are suppressed when you are detected as idle:

Suppressed When Idle Always Awards XP
Flash Images Bubble Pops (requires clicking)
Subliminal Messages Video Attention Checks (requires interaction)
Bouncing Text Lock Card (requires typing)
Bubble Count (requires answering)
Mantra (requires typing)

Rate Limits

Several activities have built-in rate limits to prevent exploit farming:

  • Bouncing Text: 2-second cooldown, 150 XP/min cap
  • Bubble Count: 3-minute cooldown between games
  • Mantra: 1.5-second minimum interval, 20 XP/min cap
  • Session XP: Base capped at 2,500
Server-Side Anti-Cheat

XP and stats are validated server-side before syncing. Abnormal values, NaN data, and impossible progression rates are automatically rejected. See the Leaderboard guide for details.
